# Stocks-Tax-Calculator
Can be used to calculate liabilities and profit on transactions of shares on the Indian Market.
 
## Features
- **2 tax levers**: Can distinguish Intraday trades from overnight ones
- **Accurate taxation**: Considers all taxes levied by the regulators and brokers
- **Logging**: Allows for optional logging of trades

## Getting Started
1. Open in BlueJ or any Java IDE
2. Run void main(String args[]) in 'Tax_window.java'
3. Enter data in all fields.
4. Pattern for dates: buy date, one space, sell date
